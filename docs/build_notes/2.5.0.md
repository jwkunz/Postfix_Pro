# Build Notes - v2.5.0

## Summary
- Bumped crate version from `2.4.0` to `2.5.0`.
- Added `EVD` operation with warning-aware behavior:
  - Stack behavior: replaces top matrix with `V` and pushes `D`.
  - Implementation uses Schur decomposition under the hood.
  - If exact diagonalization is not available, operation still returns a valid decomposition result and emits a warning message.
- Added response-level warning support:
  - `ApiResponse.warning: Option<String>`
  - Frontend now displays warning text in the status/error area for successful operations that return warnings.
- Wired `evd` through:
  - Rust backend (`src/lib.rs`)
  - API + wasm bindings (`src/api.rs`)
  - Matrix panel UI button and action map (`Post_Fix_Pro.html`)
- Updated help docs for EVD behavior and warning semantics.
- Added tests:
  - Backend test for EVD success path and warning path.
  - API test validating EVD warning propagation.

- `cargo test`
- `./scripts/build_wasm.sh`
