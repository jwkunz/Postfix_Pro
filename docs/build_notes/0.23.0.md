# Build Notes - v0.23.0

## Summary
- Bumped crate minor version from `0.22.0` to `0.23.0`.
- Updated Enter key behavior in UI action handling:
  - If entry buffer is non-empty: `Enter` pushes entry to stack (existing behavior).
  - If entry buffer is empty: `Enter` now executes `dup`.
- This applies to both keyboard `Enter` and the on-screen `Enter` button because both route through the same action handler.
