# Build Notes - v2.11.0

## Summary
- Bumped crate version from `2.10.0` to `2.11.0`.
- Matrix/vector UI organization and label updates:
  - moved `norm_p` from `Vector Operators` to `Matrix Operators`
  - renamed `Toep` button to `tpltz`
  - relabeled matrix operation buttons:
    - `HadMul` -> `H mul`
    - `HadDiv` -> `H div`
    - `MatExp` -> `exp(mat)`
    - `Herm` -> `herm`
    - `MatPow` -> `mat x^y`
  - relabeled trig/rounding buttons:
    - `to_rad` -> `to rad`
    - `to_deg` -> `to deg`
    - `rnd` -> `round`
    - `decP` -> `dec part`
  - moved `ran#` button from `Constants` to `Statistics`
- Matrix builder enhancements:
  - changed `Push I(n)` behavior to `Preset I(n)` that fills Matrix Entry with an identity matrix template (no stack push)
  - added `stack vec` button that converts all scalar stack values into one column vector matrix
- Backend/API additions:
  - added `stack_vec` calculator operation, API endpoint, wasm wrapper, and regression tests
- Updated help documentation to match panel placement, labels, and new behavior.
