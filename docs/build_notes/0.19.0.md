# Build Notes - v0.19.0

## Summary
- Bumped crate minor version from `0.18.0` to `0.19.0`.
- Reintroduced hotkeys with a minimal-risk integration approach.
- Added backend and API support for stack indexing operations:
  - `roll(count)`
  - `pick(depth)`
- Added keyboard mappings:
  - `Delete` = drop
  - `d` = duplicate
  - `s` = swap
  - `r[#]` = roll `#`
  - `p[#]` = pick `#`
- Ran tests and release build.
- Copied `index.html` to `dist/index.html`.

## Commands Run
- `cargo test`
- `cargo build --release`
- `cp index.html dist/index.html`

## Build Result
- Tests: passed (`44` tests total)
- Release build: success

## Stability Notes
- Hotkey integration did not modify wasm bootstrap/runtime initialization paths.
- Command parsing for `r[#]`/`p[#]` is isolated to keyboard handler state only.
