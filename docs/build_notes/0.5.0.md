# Build Notes - v0.5.0

## Summary
- Bumped crate minor version from `0.4.0` to `0.5.0`.
- Added matrix-capable typed dispatch to existing binary ops.
- Added matrix operations: transpose and identity creation.
- Expanded test suite for matrix arithmetic and matrix error handling.
- Ran tests and release build.
- Copied `index.html` to `dist/index.html`.

## Commands Run
- `cargo test`
- `cargo build --release`
- `cp index.html dist/index.html`

## Build Result
- Tests: passed (`20` unit tests in `src/lib.rs`)
- Release build: success

## Backend Scope Added
- `add` and `sub` now support matrix + matrix with dimension validation.
- `mul` now supports:
  - matrix * matrix
  - matrix * real
  - real * matrix
- New public methods:
  - `transpose`
  - `push_identity`
- New internal matrix helpers:
  - `matrix_add`
  - `matrix_sub`
  - `matrix_mul`
  - `matrix_scalar_mul`
  - `matrix_transpose`
  - `matrix_identity`
  - `require_same_shape`
- Type mismatch and dimension mismatch paths preserve stack state on failure.
