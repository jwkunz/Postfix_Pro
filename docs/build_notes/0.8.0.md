# Build Notes - v0.8.0

## Summary
- Bumped crate minor version from `0.7.0` to `0.8.0`.
- Implemented initial browser calculator UI in `index.html` with push-button keypad and stack display.
- Added thin JS bridge that calls Rust WASM `WasmCalculator` methods and renders returned state.
- Added keyboard support for digits, decimal, Enter, Backspace, and basic operators.
- Added runtime fallback messaging when wasm package artifacts are not present.
- Added crate library output config for wasm builds (`cdylib` + `rlib`).
- Ran tests and release build.
- Copied `index.html` to `dist/index.html`.

## Commands Run
- `cargo test`
- `cargo build --release`
- `cp index.html dist/index.html`

## Build Result
- Tests: passed (`28` tests total)
- Release build: success

## Frontend Scope Added
- New UI sections:
  - Entry/error display
  - Keypad for core operations
  - Stack panel (top-first rendering)
- JS runtime behavior:
  - Imports wasm package from `./pkg/webcalculator_backend.js`
  - Initializes `WasmCalculator`
  - Dispatches button and keyboard events to API methods
  - Parses JSON responses and updates UI state/errors
  - Disables controls when runtime is unavailable
