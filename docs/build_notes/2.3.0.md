# Build Notes - v2.3.0

## Summary
- Bumped crate version from `2.2.0` to `2.3.0`.
- Added matrix decomposition operations:
  - `QR`: computes QR decomposition and returns `Q`, `R` on stack.
  - `LU`: computes partial-pivot LU decomposition and returns `P`, `L`, `U` on stack, satisfying `P*A = L*U`.
- Wired new operations through:
  - Rust backend calculator core (`src/lib.rs`)
  - API and wasm bindings (`src/api.rs`)
  - Matrix panel UI buttons and action map (`Post_Fix_Pro.html`)
- Updated help documentation (`help.md`) for QR/LU behavior and stack output conventions.
- Added backend and API tests for both decompositions.

- `cargo fmt`
- `cargo test`
- `./scripts/build_wasm.sh`
