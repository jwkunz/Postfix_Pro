# Build Notes - v2.19.0

- Bumped crate version from `2.18.0` to `2.19.0`.

  - rounding (`round`, `floor`, `ceil`, `dec part`)
- Added matrix-elementwise regression tests in backend and API test suites.
- Updated `help.md` to document panel-wise element-wise matrix behavior.
