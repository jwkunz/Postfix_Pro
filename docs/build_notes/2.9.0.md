# Build Notes - v2.9.0

## Summary
- Bumped crate version from `2.8.0` to `2.9.0`.
- Extended Hadamard operations to support matrix-scalar element-wise behavior:
  - `HadMul`: now supports `matrix ⊙ matrix` (same shape), `matrix * scalar`, and `scalar * matrix`.
  - `HadDiv`: now supports `matrix ⊘ matrix` (same shape), `matrix / scalar`, and `scalar / matrix`.
- Kept existing shape checks and divide-by-zero protections for matrix-matrix and scalar division paths.
- Added/updated tests:
  - backend regression coverage in `hadamard_mul_and_div`
  - API regression coverage in `hadamard_ops_work_via_api`
- Updated `help.md` to document scalar-enabled Hadamard behavior.
