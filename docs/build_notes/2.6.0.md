# Build Notes - v2.6.0

## Summary
- Bumped crate version from `2.5.0` to `2.6.0`.
- Added matrix CSV import/export workflow in the GUI:
  - `Import CSV -> A`: reads a `.csv` file and loads it into Matrix A editor and size controls.
  - `Export Top CSV`: writes the top-of-stack matrix to a downloadable `matrix.csv` file.
- CSV import parser supports:
  - plain real values
  - complex literals in `(re,im)` format
  - quoted CSV cells
  - consistent column-count validation across rows
- CSV export formatting:
  - real values exported as numeric cells
  - complex values exported as quoted `(re,im)` cells
- Updated help docs to describe CSV import/export controls.
- Dist assets refreshed with the updated UI and help docs.
