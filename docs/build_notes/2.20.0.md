# Build Notes - v2.20.0

## Summary
- Bumped crate version from `2.19.0` to `2.20.0`.
- Added new matrix-builder operation `ravel` with dual behavior:
  - If top-of-stack is a non-vector matrix, `ravel` performs row-scan flattening into a long column vector (`rows*cols x 1`).
  - If top-of-stack is a vector (`Nx1` or `1xN`), `ravel` unpacks entries to stack scalars in scan order.
- Added `ravel` API + wasm export.
- Added `ravel` button to the Matrix Builder panel.
- Updated help docs to describe `ravel` behavior.
- Added backend/API regression tests for both `ravel` modes.
