# Build Notes - v0.28.0

## Summary
- Bumped crate minor version from `0.27.0` to `0.28.0`.
- Updated keyboard handling to preserve text-entry editing behavior in matrix and complex fields.
- Global calculator hotkeys are now suppressed while focus is in editable text inputs/textareas/selects/contenteditable fields.
- This enables direct typing of symbols like `-` and `.` in matrix/complex entries without triggering calculator operations.
- Matrix editing now cleanly supports explicit multiline input with `Shift+Enter`.
- Updated `help.md` keyboard notes for matrix/complex entry focus behavior.
