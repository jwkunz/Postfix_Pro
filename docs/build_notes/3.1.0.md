# Build Notes - v3.1.0

## Summary
- Bumped crate version from `3.0.3` to `3.1.0`.
- Added single-level undo support in backend/API:
  - Calculator now stores one pre-operation snapshot.
  - New `undo` operation restores the previous successful state.
  - Undo snapshot is recorded only when an operation succeeds.
  - Added API test coverage for undo behavior.
- Exposed undo through WASM bindings (`WasmCalculator::undo`).
- Updated scalar keypad layout:
  - Added `undo` button in the previous EXP position.
  - Moved `EXP` beneath `3`.
  - Moved `.` beneath `2`.
  - Moved `0` beneath `1`.
- Added help text for `undo` and kept tooltip text source in sync via `help.md`.
